Personal project portfolio site

A static portfolio site with an on-demand authoring tool that turns each project's notes into a genericized, approval-gated entry.

Eleventy (static-site generator)Nunjucks templatesMarkdownNode.jsClaude Code skill (SKILL.md)Git / GitHub Pages

The problem

I needed one place to track projects for two audiences at once — a quick status for manager updates and a polished impact summary for a resume/interview — without risking leaking work details. Ad-hoc write-ups were inconsistent and easy to over-share in.

What I built

A local-first static site fed by a two-layer content model — a public genericized entry plus a private, git-ignored learning log — authored by a custom tool that drafts both, flags anything identifying, waits for my approval, and runs a deterministic "never-publish" safety scan before anything goes live.

Impact

In progress — the site builds and previews locally, and the authoring tool and safety scan both work; publishing to free hosting is the next milestone.

My role

Designed and built the whole thing end to end — the site, the two-layer content model, the authoring tool, and the safety scan.

Built locally with Eleventy. Published deliberately.
